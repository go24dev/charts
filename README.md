Renoki Co. Helm Repo
====================

Helm Charts provided by Renoki Co. 🚀

## 🚀 Installation

To install the charts repository:

```bash
$ helm repo add nham24 https://helm.go24.app
$ helm repo update
```

To install a specific chart:

```bash
$ helm install some-release nham24/<chart>
```

## Search for available repos

```bash
$ helm search repo
```
